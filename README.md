# Cisco CyberOps Associate 200-201 — Study Repository

Personal study notes, lab write-ups, and reference material for the
Cisco Certified CyberOps Associate (CBROPS 200-201) certification.

## Why CyberOps?

My professional background sits in the ICS/OT security space, where
the day-to-day focus is on physical security operations. What I don't get much
of professionally is hands-on network security monitoring — reading
packet captures, writing Snort rules, correlating SIEM alerts, and
doing the kind of tier-1 SOC analysis work the CyberOps exam is
built around.

This certification is my structured way of closing that gap. The
networking and packet analysis skills in the exam blueprint —
Wireshark, tcpdump, NetFlow, Zeek, intrusion analysis — are areas
I want to build genuine depth in, not just familiarity.
